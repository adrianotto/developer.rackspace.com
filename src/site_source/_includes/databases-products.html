MongoDB

  • Blog Posts
  • Getting Started
  • API Reference
  • Developer Guide

MongoDB is a document database. It's native data format is BSON (similar to JSON). MongoDB is schema-less, includes constructs for high availability and has built in horizontal scaling. It has a deep community and rich driver ecosystem to work with just about any popular language.

Use cases

MongoDB is a great for projects that natively use JSON data structures, have dynamic schemas, and where ease and speed of development is paramount. It's very fast and easily scalable for a wide variety of use cases.
