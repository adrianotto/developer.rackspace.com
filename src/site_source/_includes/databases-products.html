MongoDB is a document database. It's schema-less, includes constructs for high availability and has built in horizontal scaling. It has a deep community and rich driver ecosystem to work with just about any popular language.

  • Document database (JSON/BSON)
  • Apache license
  • Runs on linux, Mac OS X
  • Atomic operations, pluggable engine architecture (2.6+), and aggregation framework
  • Mobile, gaming, and content management

Elasticsearch is a search server based on Lucene. It provides a distributed, multitenant-capable full-text search engine with a RESTful web interface and schema-free JSON documents. Elasticsearch is developed in Java and is released as open source under the terms of the Apache License.
